Structural Modeling Strict Subordination Ordering Application

Strict subordination ording (SSO) is a structural modeling method used
to structure a system in a linear order with only one object at each level
of the structure.

This web application is designed to assist in the strict ordering of an unknown
or poorly defined system. Structural modeling is used to create a system structure
from a disorganized and/or unordered set of objects. Test methods are provided to properly
test the structuring methods and techniques, a system with known order is selected
and scrambled into a ramdom system state. This ramdom, unstructured system is then
structured using the methods and techniques associated with structural modeling. When
the ordering process is complete, the final output from the process is checked against
the known system structure.

This web application supports structural modeling by organizing and presenting a web
based interface to a strict subordination matrix. This strict subordination matrix
is used to order 19 cities which have a natural language structuring relationship.
This natural language structuring relationship is "north-of." Only one city is
allowed at each level.
